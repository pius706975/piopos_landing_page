## PIOPOS Landing Page


## Installation
### 1\. Clone this Repository

```bash
git clone https://github.com/pius706975/piopos_landing_page.git
```

### 2\. Navigate to the directory

```
cd your-directory
```

### 3\. Install dependencies

```
npm install or npm i
```

### 5\. Run Dev Server

```
npm run dev
```
